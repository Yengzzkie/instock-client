## How to Use
1. Clone the repository:
    ```bash
    git clone git@github.com:Yengzzkie/instock-client.git
    ```
2. Install dependencies:
    ```bash
    npm install  
    ```
3. Run a local development server:
    ```bash
    npm run dev
    ```
